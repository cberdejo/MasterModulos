<p> En este módulo se presentan los fundamentos del <strong>aprendizaje automático (Machine Learning)</strong>, sus principales paradigmas y algoritmos utilizados para el análisis de datos. Se exploran modelos de aprendizaje <strong>supervisado y no supervisado</strong>, técnicas de <strong>reducción de dimensionalidad</strong> y métodos de <strong>combinación de modelos (ensembles)</strong>.</p>

<h2 id="mod8presentacion"> 📚 Presentaciones </h2>
Se pueden encontrar documentos que sirven como primera aproximación al tema de Machine Learning. 
<h3>🧠 Introducción al Aprendizaje Automático</h3>
<p>El módulo comienza con una introducción general al <strong>Machine Learning</strong>, definiéndolo como un proceso en el que los algoritmos aprenden patrones de los datos para hacer predicciones o tomar decisiones sin ser programados explícitamente.</p>

<ul>
    <li><strong> Data Mining y Machine Learning:</strong> Relación con estadística e inteligencia artificial.</li>
    <li><strong> Paradigmas de aprendizaje:</strong> Supervisado (clasificación y regresión), no supervisado (clustering, reducción de dimensionalidad) y aprendizaje por refuerzo.</li>
    <li><strong> Importancia del modelado:</strong> Cómo las máquinas pueden aprender observando datos y generalizar sobre nuevos casos.</li>
</ul>

<h3>📊 Aprendizaje Supervisado</h3>
<p>Se explican los algoritmos más utilizados para <strong>aprendizaje supervisado</strong>, donde se entrenan modelos con datos etiquetados:</p>
<ul>
    <li> <strong>Clasificación:</strong> Modelos que asignan categorías a los datos (ejemplo: spam vs no spam).</li>
    <li> <strong>Regresión:</strong> Modelos que predicen valores numéricos (ejemplo: precios de vivienda).</li>
</ul>
<p><strong>Ejemplo de algoritmos:</strong></p>
<ul>
    <li> Regresión logística</li>
    <li> Árboles de decisión</li>
    <li> Máquinas de soporte vectorial (SVM)</li>
    <li> Redes neuronales artificiales</li>
</ul>

<p>También se discuten temas clave como:</p>
<ul>
    <li> <strong>Sobreajuste y subajuste:</strong> Cómo evitar modelos que memoricen los datos de entrenamiento sin generalizar bien.</li>
    <li> <strong>Validación cruzada y ajuste de hiperparámetros.</strong></li>
</ul>

<h3>📊 Aprendizaje No Supervisado</h3>
<p>Se presentan métodos donde los datos no tienen etiquetas y el objetivo es encontrar patrones ocultos:</p>
<ul>
    <li> <strong>Clustering:</strong> Agrupación de datos en categorías similares.</li>
    <li> <strong>Reducción de dimensionalidad:</strong> Métodos para simplificar los datos conservando la información relevante.</li>
</ul>
<p><strong>Ejemplo de algoritmos:</strong></p>
<ul>
    <li> K-Means</li>
    <li> DBSCAN</li>
    <li> PCA (Análisis de Componentes Principales)</li>
    <li> LDA (Análisis Discriminante Lineal)</li>
</ul>
<p>Estos métodos son utilizados en segmentación de clientes, detección de anomalías y análisis exploratorio.</p>

<h3>🏆 Métodos de Ensamble</h3>
<p>Se estudian técnicas para mejorar la precisión combinando múltiples modelos:</p>
<ul>
    <li> <strong>Bagging (Bootstrap Aggregating):</strong> Como el <strong>Random Forest</strong>, que entrena múltiples árboles de decisión y promedia sus resultados.</li>
    <li> <strong>Boosting:</strong> Métodos como <strong>AdaBoost</strong> y <strong>Gradient Boosting</strong>, que ajustan modelos secuenciales para mejorar el rendimiento.</li>
    <li> <strong>Stacking:</strong> Combinación de varios modelos con un meta-modelo final.</li>
</ul>
