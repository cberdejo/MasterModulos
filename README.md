<h1 id="mod4">🗄️ 4: Gestión de datos en el Big Data, Bases de Datos NoSQL</h1>
<p>📌 En este módulo exploraremos cómo se gestionan los datos en entornos Big Data utilizando bases de datos NoSQL. Se abordarán dos tecnologías clave: <strong>MongoDB</strong> y <strong>Cassandra</strong>. Aprenderás los conceptos fundamentales, operaciones básicas y técnicas avanzadas.</p>

<h2 id="mongodb">🍃 MongoDB</h2>
<h3 id="mod4diapo">📑 Temario</h3>
<ul>
    <li><strong>📌 Introducción a MongoDB:</strong> Explicación del modelo de documentos, JSON/BSON y comparación con bases de datos relacionales.</li>
    <li><strong>📂 CRUD en MongoDB:</strong> Operaciones básicas de <code>insert</code>, <code>find</code>, <code>update</code> y <code>delete</code>.</li>
    <li><strong>🔎 Índices en MongoDB:</strong> Uso de índices para optimizar la búsqueda de datos.</li>
    <li><strong>📊 Agregaciones:</strong> Uso del pipeline de agregaciones para consultas avanzadas.</li>
    <li><strong>📈 MongoDB Charts:</strong> Visualización de datos mediante herramientas de gráficos.</li>
</ul>

<h3 id="mod4tareas">📝 Tareas</h3>
<ul>
    <li><strong>📊 Consultas con <code>find</code> y agregaciones:</strong> Implementación de búsquedas avanzadas con operadores como <code>$match</code>, <code>$group</code> y <code>$project</code>.</li>
    <li><strong>💰 Conversión de precios:</strong> Transformación de datos de precios en diferentes monedas utilizando agregaciones.</li>
    <li><strong>🔗 Join entre colecciones:</strong> Uso de <code>$lookup</code> para combinar datos de distintas colecciones.</li>
</ul>

<h2 id="cassandra">🔷 Cassandra</h2>
<h3 id="mod4diapocassandra">📑 Temario</h3>
<ul>
    <li><strong>📌 Introducción a Cassandra:</strong> Explicación del modelo de datos basado en columnas y comparación con MongoDB.</li>
    <li><strong>🛠️ Estructura de datos:</strong> Explicación de <code>keyspaces</code>, <code>column families</code> y <code>partitions</code>.</li>
    <li><strong>🔎 Consultas básicas:</strong> Uso de <code>SELECT</code>, <code>INSERT</code> y <code>UPDATE</code> en Cassandra Query Language (CQL).</li>
</ul>

<h3 id="mod4tareasCassandra">📝 Tarea</h3>
<ul>
    <li><strong>📄 Consultas básicas:</strong> Práctica con consultas de inserción y recuperación de datos en Cassandra.</li>
</ul>

<h2 id="temario5">Temario</h2>
<h4>⭐ Modelado Estrella</h4>
<p>Esquema utilizado en <strong>Data Warehouses</strong> donde una <em>tabla hecho</em> almacena métricas y se conecta a <em>tablas dimensión</em> que describen los datos.</p>


<h4 id="etl">🔄 ETL (Extract, Transform, Load)</h4>
<p>Proceso para integrar datos desde múltiples fuentes:</p>
<ul>
    <li><strong>Extract:</strong> Recuperar datos.</li>
    <li><strong>Transform:</strong> Limpiar y dar formato.</li>
    <li><strong>Load:</strong> Cargar en un Data Warehouse.</li>
</ul>



<h4 id="data-profiling">📊 Data Profiling</h4>
<p>Proceso de <strong>análisis de datos</strong> para evaluar su calidad y estructura.</p>


<h1 id="mod6">⚡ 6: Procesamiento de Datos Escalable con Spark y Hadoop</h1>
<p>🚀 En este módulo aprenderás sobre procesamiento de datos a gran escala con **Apache Spark** y **Hadoop**. Exploraremos el ecosistema de ambas tecnologías, cómo utilizarlas para manejar Big Data y cómo ejecutar consultas y análisis eficientes.</p>

<h2 id="mod6temario">📚 Temario</h2>

<h3 id="spark">🔥 Apache Spark</h3>
<ul>
    <li><strong>📌 Introducción:</strong> Qué es Spark y cómo se diferencia de otros frameworks de procesamiento de datos.</li>
    <li><strong>🔗 Arquitectura:</strong> Explicación de los conceptos clave: **RDDs, DataFrames, Spark SQL**.</li>
    <li><strong>💡 APIs de Spark:</strong>
        <ul>
            <li><strong>RDD API:</strong> Operaciones sobre datos distribuidos, transformaciones y acciones.</li>
            <li><strong>DataFrame API:</strong> Estructuras tabulares, consultas optimizadas y comparación con bases de datos.</li>
        </ul>
    </li>
    <li><strong>⚙️ Entornos de ejecución:</strong> Spark en **local, cluster (YARN, Kubernetes, Mesos)**.</li>
</ul>

<h3 id="hadoop">🐘 Apache Hadoop</h3>
<ul>
    <li><strong>📌 Introducción:</strong> Explicación de Hadoop como un sistema de procesamiento distribuido.</li>
    <li><strong>📁 HDFS (Hadoop Distributed File System):</strong> Almacenamiento distribuido de grandes volúmenes de datos.</li>
    <li><strong>🗂️ Hadoop YARN:</strong> Gestión de recursos y ejecución de tareas en clústeres.</li>
    <li><strong>🔄 MapReduce:</strong> Modelo de programación para procesamiento paralelo de datos.</li>
</ul>

<h3 id="spark_tareas">🔥 Spark</h3>

    <li><strong>💻 Proyecto Python-Spark:</strong> Desarrollo de una aplicación en **PySpark** para análisis de datos.</li>
    <li><strong>☕ Proyecto Java-Spark:</strong> Implementación de un programa en **Java** para procesamiento distribuido en Spark.</li>
